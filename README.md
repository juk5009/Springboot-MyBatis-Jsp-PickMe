# Team 프로젝트

## 기술스택
- JDK 11
- Springboot 2.7.9
- MyBatis
- 테스트 h2 디비
- 프로덕션 MySQL 디비
- JSP

## 모델링
### 1단계
- User
- Company
### 2단계
- Notice
- Apply
- Skill
### 3단계
- Companyinfo

## 기능정리
### 1단계 
- 회원가입, 로그인, 이력서쓰기, 이력서 상세보기, 채용 공고 쓰기, 채용공고 상세보기, 공고 지원
### 2단계 
- 회원정보(수정, 삭제), 채용 공고(수정, 삭제), 이력서(수정, 삭제)
### 3단계
- 관심 매칭 서비스 , 구직자 추천 서비스
